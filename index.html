Haunted Hotspots

  • View Project
  • View on Github

Haunted Hotspots is a website where users can share and comment on one another's haunted places they've visited or heard about. I did this project to learn more about the back-end of web development as well as continue practicing my front end skills. I feel that at least being familiar with the backend will strengthen my front-end skills that much more.

I learned about RESTful routing and databases, user authentication and authorization. A user has to create an account in order to be able to add or comment on an existing haunted place. The user is also only allowed to edit one's own comments and haunted places.

By using jQuery plugins and node modules I was able to learn how to take others' modules and use them to serve my needs in creating this website.

Also, when visiting the site, don't mind the user EnlightenedSkeptic. He's very close to getting banned.

Cat Clicker

  • View Project
  • View on Github

This project records the number of times each cat has been clicked. You can also click the admin button to manually change each cat's name, image, or the number of times it has been clicked. Try changing an image to "images/cat6.jpg" or "images/cat7.jpg".

I specifically did not use jQuery so I could have a more in depth appreciation of DOM manipulation and what jQuery does when I do use it.

This was my first project using a model-view-controller format for my javascript, making this app scalable. If you wanted to have 50 or even 500 cats instead of 5 you could add them easily.
